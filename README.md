# Infinite Parkour Datapack Installer
An updater for the Infinite Parkour Datapack 

# Use Instructions 
install git
[Git for Windows](https://git-scm.com/downloads/win
)

If you use the normal .minecraft folder you don't need to do anything
If you use a different location, make a config.txt and put the path the the folder (note. not the world folder)
